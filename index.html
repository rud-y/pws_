    Vet management app


    An application for managing information about incoming 'pet patients', their owners, and a veterinary practice employees. A user can create, edit and delete details about vets, customers, pets and appointments. Owners can arrange appointment for their pet with one of the vets. Check it out here or watch a short demo.


    Languages/frameworks used:
    MVC architecture pattern utilising Ruby and Sinatra web framework for request routing and PostgreSQL for storing and editing user input data.

    Tournament manager


    Tournament manager application is designed mainly for sports enthusiasts. It was built as a pair project with my former CodeClan buddy Robert Peacock without whom the project would not exist as it is today. Huge credit goes to him :). A user can manage different tournaments by adding new tournaments, teams and creating new matches: fixtures, then results. After match score is submitted the correspondent league table is updated with wins, draws or losses and the teams in a table sorted according to points earned.
