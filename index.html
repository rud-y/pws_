    Vet management app

    An application for managing a small veterinary practice. A user is able to create, edit and delete pets, customers and vets. Appointments can be arranged for certain vet and a pet patient.

    Tournament manager

    Tournament manager application designed not only for sports enthusiasts. A user can manage different tournaments by adding new tournaments, teams and creating new matches: either fixtures or results. After match is submitted then the correspondent league table is updated and the teams in it sorted according to points they earned.

    Dobble - browser game

    Dobble app is a browser adaptation of a card game Dobble.
